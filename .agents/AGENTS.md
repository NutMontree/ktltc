# KTLTC Project Guidelines

- **โปรเจกต์**: ระบบบริหารจัดการข่าวสารและข้อมูลวิทยาลัยเทคนิคกันทรลักษ์ (KTLTC)
- **Tech Stack**: Next.js (App Router), React 19, Tailwind CSS v4, Ant Design, HeroUI
- **Database / Auth**: MongoDB (Mongoose), Next-Auth
- **ข้อกำหนดในการเขียนโค้ด**: 
  - ให้รักษาโครงสร้างและ Provider เดิม เช่น `ThemeProvider`, `ActiveUserTracker`, `NextAuthProvider`
  - ใช้ Tailwind CSS เป็นหลักในการจัดการสไตล์
  - ให้ระมัดระวังเกี่ยวกับคอมเมนต์เดิมและการทำงานของฟังก์ชันที่เชื่อมต่อกับ UI หลัก
- **เป้าหมาย**: เพื่อให้ AI ทุกตัวที่เข้ามาทำงานในแชตใหม่ได้อ่านและรับรู้บริบทของโปรเจกต์ก่อนเริ่มงาน
